#RATcage
Ruby gem that helps to generate  [Rails Application Templates](http://guides.rubyonrails.org/rails_application_templates.html) (RAT)

###Usage:
####Install:
    gem install ratcage

####Gemfile:
    gem 'ratcage', '~> 0.0.2'
